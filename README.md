# cmaj
Music theory from the command line :D

## Examples


### Scales

View major/minor, sharp/flat scales

`cmaj --scale=Em`

`E F# G A B C D`

`cmaj --scale=Bbm`

`Bb C Db Eb F Gb Ab`

`cmaj --scale=C`

`C D E F G A B `

### Chords

View a chord. 
At the moment only major and minor triads are supported but others (M7, m7, M9, m7, 7b9 etc...) are coming.
`cmaj --chord=C`

`C E G`

`cmaj --chord=Em`

`E G B`
